Rakmo Banking Application is a secure and scalable Spring Boot RESTful Banking API that provides core banking functionalities such as account creation, deposits, withdrawals, and customer information retrieval.
The application is built using Spring Boot, Spring Security with JWT authentication, and MySQL, and follows best practices including custom response handling, global exception management, and RESTful design principles.

This project demonstrates real-world backend development concepts such as authentication & authorization, database integration using JPA/Hibernate, and clean layered architecture, making it suitable for enterprise-grade applications and backend development learning.
